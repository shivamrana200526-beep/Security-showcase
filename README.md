  ## Features

  - Search by title, student name, or keyword
  - Filter by category (Text, NLP, Image, Forensics, Audio, Deep Learning, Video)
  - Copy project link button on each card
  - Responsive grid layout (2 columns desktop, 1 column mobile)
  - Smooth hover animations and glassmorphism design

  ## Usage

  Open `index.html` directly in any browser — no build step or server required.
